Welcome to Food Tracker

Home page

  • When not logged in only displays this information
  • When logged in shows this info, last 5 food logs, and overview of last 5 days

Foods

  • Lists Foods with their category and type
  • Categories can be
    • Restaurant
    • Fast Food
    • Commercial
    • Homemade
    • Raw
    • Generic
  • Types is user generated. There are some base food items for users to start with
  • When A is click a dropdown of subtypes are show. For instance the Apple type has Fuji, Gala, Grannysmith, Honeycrisp, and Red Delicious subtypes
  • Click on a subtype brings you to it's food page which shows all it's servings
  • The search bar allows you to search in type and subtype for a match
  • Loggedin Features
    • You are able to create a new food that's only available to you
    • Createing a food brings you to create a serving
    • With Subtypes open you are able to Add it to your food log.
    • The subtype food page allows you to add a new serving

Prediction

  • The prediction page allows you to input your stats and will give you a rundown of when you'll reach your goal weight as well as the amount of calories you would eat on a given day
  • When logged in the stats will be auto input from your profile
  • After hitting the predict button the inputs will close to allow you to view the prediciton table

Food Log

  • Shows Food logs created
  • Can edit logs
  • Can copy logs and the date will be the current date based of UTC.(I need to fix that)

Daily

  • Shows Aggregate values for all food logs eaten that date
  • Can select an activity level for that day (effects the calculations)
  • Can add an actual weight for that day. (does not effect calculations)
  • The "A" will create a dropdown with addtional overview data as well as all food logs for that date

Register

  • Creates an account and logs you in
  • After successful creation will be brought to create profile page
  • If you don't create a profile going to any page should redirect to create a profile
  • Registering will send an email from techgnome@gmail.com to verify the account
  • If you do not verify the account a card will be displayed to resend the verify email

Profile

  • Shows email and starred out password(no password info is sent) and the ability to edit them
  • Show profile information and ability to edit it
